# Contributing

Thanks for considering a contribution to `codex-token-guard`.

## Local Checks

```bash
npm test
node ./src/cli.js scan
node ./src/cli.js brief "improve maintainer workflow"
```

## Contribution Scope

Good first contributions:

- Improve noisy file detection.
- Improve relevance scoring.
- Add fixture-based tests.
- Improve maintainer workflow examples.
- Add JSON output for CI integrations.

Please keep the default workflow local-only and avoid external API calls.

## Claims

Do not describe this project as measuring exact Codex billing or internal token usage. It provides local estimates and scope guidance.
